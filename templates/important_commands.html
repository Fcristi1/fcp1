Important Commands

    Virtual environment setup:
    python3 -m venv mqtt_env
    source mqtt_env/bin/activate
    pip install flask flask-socketio paho-mqtt
    python flask1.py

    Ngrok setup:
    ngrok http 5000

    Tmux setup:
    tmux
    python3 webserver.py
    ctrl + b -> c (new window)
    exit (to exit tmux)
